## 🍿 내 방 사진 한 장이면 충분해요

내 방 사진을 올리면, AI가 방 분위기를 읽고 **세상에 하나뿐인 영화 티켓**을 만들어 줘요.

- 🎞️ 이 방에 어울리는 **영화 제목, 장르, 태그라인**을 지어주고
- 🧑 이 방에 살고 있을 **주인공의 성격과 이야기**를 상상해 주고
- 🎥 이 영화에서만 볼 수 있는 **명장면 3가지**까지 만들어 줘요

AI가 내 방을 읽고, 나를 해석해 주는 경험 — 한번 해보시면 은근 빠져들거예요.

> 완성된 티켓은 이미지로 저장할 수 있어요!

<br>

## 📸 사용 방법

정말 간단해요. 3단계면 끝!

```
 1️⃣  사이트에 접속한다
 2️⃣  방 사진을 올린다
 3️⃣  [티켓 발권하기] 를 누른다
```

10초 정도 기다리면 나만의 영화 티켓이 완성돼요.

> 💡 같은 방이라도 사진 각도나 조명에 따라 전혀 다른 영화가 나올 수 있어요. 여러 번 해보는 것도 재미있답니다!

## 🙋 자주 묻는 질문

**Q. 사진은 어디에 저장되나요?**

어디에도 저장되지 않아요. 분석이 끝나면 바로 사라지니 안심하세요.

**Q. 꼭 방 사진이어야 하나요?**

아니요! 다른 사진도 결과는 나와요. 다만 방 사진일 때 가장 재미있는 결과가 나와요.

**Q. 결과가 맘에 안 들면요?**

하단의 [다른 방 사진으로 다시 해보기] 버튼을 눌러 보세요. 같은 사진이어도 매번 다른 영화가 만들어져요.
