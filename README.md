# 🎬 내 방이 영화가 된다면?

> 방 사진 한 장이면 충분해요. AI가 당신의 방을 영화로 만들어 드립니다.

<br>

## ✨ 지금 바로 해보세요

👉 **https://ohouse-movie.vercel.app**

준비물은 딱 하나, **방 사진 한 장**이에요.

<br>

## 🍿 이런 거예요

집에서 뒹굴뒹굴하다 문득 이런 생각 해본 적 없나요?

*"내 방이 영화 배경이면... 제목이 뭘까?"*

그 쓸데없지만 소중한 궁금증을 해결해 드립니다.

사진을 올리면 AI가 분석해서 나만의 **영화 티켓**을 만들어 줘요.

- 🎞️ 방 분위기에 어울리는 **영화 제목과 장르**
- 🎭 이 방에서 느껴지는 **분위기 키워드**
- 🧑 이 방에 사는 **주인공의 성격과 이야기**
- 🎥 이 영화에서 볼 수 있는 **명장면 3가지**

> 완성된 티켓은 이미지로 저장할 수 있어요. 친구한테 보내서 자랑해 보세요.

<br>

## 📸 사용 방법

정말 간단해요. 3단계면 끝!

```
 1️⃣  사이트에 접속한다
 2️⃣  방 사진을 올린다
 3️⃣  [티켓 발권하기] 를 누른다
```

끝이에요. 나머지는 AI가 알아서 해줍니다.

10초 정도 기다리면 당신의 방이 영화가 되어 돌아와요.

> 💡 같은 방이라도 사진 각도나 조명에 따라 전혀 다른 영화가 나올 수 있어요. 여러 번 해보는 것도 재미있답니다!

<br>

## 🙋 자주 묻는 질문

**Q. 사진은 어디에 저장되나요?**

어디에도 저장되지 않아요. 분석이 끝나면 사라집니다. 안심하세요.

**Q. 아무 사진이나 되나요?**

방 사진이 아니어도 결과는 나와요. 다만 방 사진일 때 가장 재미있는 결과가 나옵니다.

**Q. 결과가 맘에 안 들면?**

[다른 방 사진으로 다시 해보기] 버튼을 눌러보세요. 같은 사진이라도 매번 다른 영화가 나와요.
